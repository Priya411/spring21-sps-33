Political Contributions Tracker

Welcome to the prototype of the Political Contributions Tracker.

This project uses Google Firestore and the OpenFEC API.
All data shown is populated using real data provided by the FEC.
Use the search bar below to bring up a specific person's contributions chart page. In a future update, the search feature will dynamically alter the names shown in the table on the main page.
